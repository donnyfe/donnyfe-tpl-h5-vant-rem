# donnyfe-tpl-h5-vant-rem

技术栈：Vue3、Vant、UnoCSS、VueRouter、Pinia、Pinia-plugin-persistedstate、axios、

## 项目特性

- 使用Vue3进行开发
- 使用Vant组件库
- 使用UnoCSS进行样式管理
- 使用rem单位，适配移动端
- 使用VueRouter进行路由管理
- 使用Pinia进行状态管理
- 使用Pinia-plugin-persistedstate进行状态持久化
